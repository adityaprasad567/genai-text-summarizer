# 🧠 AI Text Summarizer

A lightweight extractive text summarization web application built using **Python, NLTK, and Streamlit**.

This project generates concise summaries from long text by selecting the most important sentences using frequency-based scoring.

---

## 🚀 Live Demo

Deployed on Streamlit Cloud.

---

## 📌 Features

- Extractive text summarization
- Frequency-based sentence scoring
- Adjustable summary length
- Word count display
- Download summary as text file
- Lightweight and optimized for low memory environments (≤ 500MB RAM)

---

## 🛠️ Technologies Used

- Python
- NLTK (Natural Language Toolkit)
- Streamlit

---

## 🧠 How It Works

1. Text preprocessing (cleaning & normalization)
2. Sentence tokenization
3. Word frequency calculation
4. Sentence scoring
5. Selection of top-ranked sentences
6. Display summarized output

This system performs **extractive summarization**, meaning it selects important sentences from the original text rather than generating new ones.
